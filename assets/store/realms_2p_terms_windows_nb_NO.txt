Realms er et abonnement på personlig flerspillerserver som brukes med støttede versjoner av Minecraft (spillet selges separat) på Windows og andre støttede plattformer du eier spillet på (Minecraft Java Edition støttes ikke). Du kan si opp abonnementet for å stoppe fremtidige betalinger ved å gå til Tjenester og abonnementer i Microsoft-kontoen din. Hvis abonnementet ditt er inaktivt i mer enn 18 måneder, vil Realms-dataene bli slettet fra serveren. Kostnader fra Internett-leverandør kan påløpe. Realms-rabatter kan ikke kombineres med andre tilbud og kan ikke innløses for kontanter: krav, funksjoner, nettjenester eller abonnement kan endres og/eller bli trukket tilbake. Underlagt Microsofts tjenesteavtale (microsoft.com/msa) og Realms lisensavtale for sluttbrukere (https://www.minecraft.net/realms/terms/).
