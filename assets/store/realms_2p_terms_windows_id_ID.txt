Realms adalah langganan untuk konten dan server multipemain pribadi untuk digunakan dengan versi Minecraft yang didukung (permainan dijual terpisah) di Windows dan platform lain yang didukung di mana tempat kamu memiliki permainannya (Minecraft Java Edition tidak didukung). Kamu bisa membatalkan langgananmu untuk menghindari perubahan di masa mendatang dengan mengakses Layanan & langganan Akun Microsoft kamu. Jika langgananmu tidak aktif selama lebih dari 18 bulan, data Realms kamu akan dihapus dari server. Biaya ISP mungkin berlaku. Diskon Realms tidak bisa digabungkan dengan penawaran lain dan tidak bisa ditukar dengan uang tunai. Persyaratan, fitur, layanan online, atau langganan dapat berubah dan/atau berakhir. Taati Perjanjian Layanan Microsoft (microsoft.com/msa) dan Perjanjian Lisensi Pengguna Akhir Realms (https://www.minecraft.net/realms/terms/).
